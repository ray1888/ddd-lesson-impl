# 这个项目的是为把自己抽象出来的模型

## 这个模型涉及的对象
1. 学生
2. 老师
3. 课程
4. 订单

## 业务上面需要提供的功能
1. 安排上课
2. 查看全部课程
3. 学生查看自己的课程
4. 老师查看自己的课程
5. 修改课程时间
6. 获取学生列表
7. 获取老师列表
8. 获取单个老师的信息
9. 获取单个学生的信息
10. 获取所有订单
11. 获取某个学生的订单
12. 业务规则- 如果学生有三节课再购买新订单，可以打8折


## 目录结构
```
|- src  // 代码存放位置  
    |- domain  领域  
        |- model 领域模型  
            |- read 读取的领域模型  
        |- repository 持久化接口  
        |- service   领域服务接口  
    |- infrastructure 基础  
        |- mapper 数据库表映射  
```
